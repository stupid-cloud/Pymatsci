.. _installation:

Installation
============

1.To use pymatsci, first install pymatgen and numpy using pip:

.. code:: console

   pip install numpy
   pip install pymatgen

2.Install pymatsci

.. code:: pyth

   pip install pymatsci

In addition, you can also visit `pypi`_ to download.

.. _pypi: https://pypi.org/project/pymatsci/
